# 🌱 フォーカス・畑 (Focus Garden)

集中するたび、畑に植物が育つ**ポモドーロタイマー**です。ビルド不要・依存ゼロ、1枚の HTML で動くオフライン対応の PWA です。

## 特長

- ⏱ **ポモドーロタイマー** — 集中 25 分 → 休憩 5 分のサイクル
- 🪴 **育つ庭** — 集中を 1 回終えるたびに、ランダムな植物が 1 本増えます
- 🥀 **しおれる芽** — 集中中にタブを離れたり一時停止すると、育ちかけの芽はしおれ、連続記録（streak）はリセットされます
- ⏰ **今日のタイムライン** — その日いつ集中したかを 24 時間軸で表示
- 📊 **週間チャート** — 過去 7 日間の集中本数をグラフ化
- 🏆 **総合記録** — 合計本数・合計時間・連続記録ベスト
- 🔥 **時間帯ヒートマップ** — 集中しがちな時間帯を可視化
- 📲 **PWA** — ホーム画面に追加でき、オフラインでも動作

## 使い方

`http(s)` か `localhost` 経由で開いてください（Service Worker は `file://` では動かないため）。

```bash
# このフォルダで簡易サーバーを起動
python3 -m http.server 8000
```

ブラウザで <http://localhost:8000/> を開きます。

## データについて

記録はすべて**お使いのブラウザの `localStorage`** に保存されます。サーバーには何も送信されません。「記録をリセット」ボタン、またはブラウザのデータ消去でクリアできます。

## ファイル構成

| ファイル | 役割 |
|----------|------|
| `index.html` | アプリ本体（UI・タイマー・各種グラフ描画） |
| `sw.js` | オフライン用 Service Worker（キャッシュ管理） |
| `manifest.json` | PWA マニフェスト（アプリ名・アイコン・テーマ色） |
| `icon.svg` | アプリアイコン |
